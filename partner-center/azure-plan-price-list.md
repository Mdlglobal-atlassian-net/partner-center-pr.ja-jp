## <a name="see-pricing-for-subscriptions-under-the-azure-plan-pricing"></a><span data-ttu-id="b05be-108">Azure プランのサブスクリプションに対する価格を確認する</span><span class="sxs-lookup"><span data-stu-id="b05be-108">See pricing for subscriptions under the Azure plan pricing</span></span>

1. <span data-ttu-id="b05be-109">左側のパートナー センターのメニューから、 **[販売]** を選択し、 **[マーケットプレース]** を選択します。</span><span class="sxs-lookup"><span data-stu-id="b05be-109">From the Partner Center menu on the left, select **Sell** and then select **Marketplace**.</span></span>

2. <span data-ttu-id="b05be-110">**[エクスポートの種類]** で、 **[Azure プランの従量課金価格]** を選択します。</span><span class="sxs-lookup"><span data-stu-id="b05be-110">Next to **Export type**, select **Azure plan consumption pricing**.</span></span>

3. <span data-ttu-id="b05be-111">**[日付での価格]** で、目的の日付を選択します (例: **[現在]** )。</span><span class="sxs-lookup"><span data-stu-id="b05be-111">Next to **Pricing for date**, select the date you want, for example, **Current**.</span></span> <span data-ttu-id="b05be-112">注: **[外国為替レート]** を選択して、現在の外国為替レートをエクスポートすることもできます。</span><span class="sxs-lookup"><span data-stu-id="b05be-112">Note: you can also select **FX rates** to export the current FX rates.</span></span>

![Azure の価格 2](images/azure/pricelist2.jpg)

4. <span data-ttu-id="b05be-114">**[マーケットプレース]** で、製品の **[種類]** と **[カテゴリ]** を選択するか、製品を検索します。</span><span class="sxs-lookup"><span data-stu-id="b05be-114">Under **Marketplace** select the **Type** and **Category** of product or search for a product.</span></span> <span data-ttu-id="b05be-115">検索条件に基づいて利用可能な製品が表示されます。</span><span class="sxs-lookup"><span data-stu-id="b05be-115">Available products based on your search are displayed.</span></span>

![pricing](images/azure/Azurepricelist1.jpg)

5. <span data-ttu-id="b05be-117">次に、 **[Azure プランの価格表のエクスポート]**  を選択して、選択した製品用の Azure プランの価格をダウンロードします。</span><span class="sxs-lookup"><span data-stu-id="b05be-117">Then select **Export Azure plan price list** to download the Azure plan prices for the product(s) you select.</span></span>


![価格表のエクスポート](images/azure/pricelist1.png)

# <a name="support-from-microsoft"></a>マイクロソフトによるサポート

**適用対象**

-  パートナー センター
-  米国政府機関向け Microsoft Cloud のパートナー センター
-  Microsoft Cloud ドイツのパートナー センター

Microsoft サービスやパートナー センターで問題が発生した場合は、Microsoft にエスカレートしてサポートを依頼し、解決まで問題を追跡できます。

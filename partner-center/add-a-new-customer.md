**<span data-ttu-id="26329-112">新しい顧客の追加</span><span class="sxs-lookup"><span data-stu-id="26329-112">Add a new customer</span></span>**

1.  <span data-ttu-id="26329-113">**[ダッシュボード]** メニューで **[顧客]** を選びます。</span><span class="sxs-lookup"><span data-stu-id="26329-113">From the **Dashboard** menu, select **Customers**.</span></span> <span data-ttu-id="26329-114">**[顧客の追加]** を選びます。</span><span class="sxs-lookup"><span data-stu-id="26329-114">Then select **Add customer**.</span></span>
2.  <span data-ttu-id="26329-115">**[アカウント情報]** ページで、会社の詳細と第一連絡先担当者を入力し、**[次へ: サブスクリプション]** を選びます。</span><span class="sxs-lookup"><span data-stu-id="26329-115">On the **Account info** page, complete the company details and primary contact information, and then select **Next: Subscriptions**.</span></span>
3.  <span data-ttu-id="26329-116">**[サブスクリプション]** ページで、正しいプランを選び、ライセンス数を入力して、**[次へ: レビュー]** を選びます。</span><span class="sxs-lookup"><span data-stu-id="26329-116">On the **Subscriptions** page, choose the correct offer(s), enter the number of licenses, and then select **Next: Review**.</span></span>

5.  <span data-ttu-id="26329-118">確認画面で確認を行い、管理者のユーザー アカウントとパスワードをコピーしたら、**[完了]** を選びます。</span><span class="sxs-lookup"><span data-stu-id="26329-118">After you’ve reviewed the confirmation, and copied the admin user account and password, select **Done**.</span></span>

<span data-ttu-id="26329-119">サブスクリプションへのアドオン ライセンスの追加など、顧客のデータを後で確認するには、「[新しいサブスクリプションの作成](create-a-new-subscription.md)」で詳細をご覧ください。</span><span class="sxs-lookup"><span data-stu-id="26329-119">To review the customer data later, including adding add-on licenses to subscriptions, see the details at [Create a new subscription](create-a-new-subscription.md).</span></span>
